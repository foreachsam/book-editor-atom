Atom Editor 使用筆記

command atom

開啟檔案

執行下面的指令，開啟某個檔案。「file_path」請更改成你要開啟的檔案名稱，或是檔案路徑。

$ atom file_path

開啟資料夾

執行下面的指令，開啟某個資料夾。「dir_path」請更改成你要開啟的資料夾名稱，或是資料夾路徑。

$ atom dir_path

這個會起動「atom」，然後在左方的樹狀瀏覽，就會以這個「資料夾」當成「根部」來展開，在這個資料夾底下的所有檔案資料夾都會顯示在左方的檔案樹狀瀏覽。 也就是把這個資料夾當成「project」的根目錄。

所以當你在某個資料夾底下，就可以執行下面的指令，來開啟該資料夾。

$ atom ./
